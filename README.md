# Simple Quicks Skill Challenge for Simpul Tech
Project ini dibuat untuk menyelesaikan ujian teknis untuk melamar di Simpul Technologies sebagai Frontend Engineer.

## Dependecies

- React + Vite
- Tailwind CSS
- Material UI
- React Spinners

### Start the project

```sh
cd simple-quicks
npm run dev
```
### Features
- Fitur Action Button
- Fitur Chat
- Fitur Conversation
